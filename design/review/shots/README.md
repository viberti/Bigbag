---
**Como capturar rápido**
- **Telemóvel:** abre cada ecrã na app e tira print (como tens feito). PNG de preferência.
- **Desktop (/admin, /explorar):** abre cada aba no browser e captura (print do ecrã ou DevTools → "Capture screenshot"). Não precisa de recorte fino.

**Entregar ao designer:** zipa a pasta `design/review/` inteira (HTML + `shots/`) — abre em qualquer browser, com snapshot + explicação por ecrã.
